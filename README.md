# ⚽ Gyro Soccer Arena

スマホのブラウザでサクッと遊べる、**ロケットリーグ風 3D アリーナサッカー**。  
ジャイロセンサーで車を操作して、ロケットブースト付きの車でボールを蹴り合うオンラインマルチプレイ対戦ゲームです。

> 兄弟プロジェクト [`racegame`](https://github.com/guttyanneruuuuuu/racegame) のジャイロ入力 / 自動キャリブレーション / 車体メッシュをそのまま移植し、サッカー用に物理 / アリーナ / ネットワーク / カメラを新規実装しています。

---

## 🎮 主な特徴

- 🏟️ **ロケットリーグ実寸アリーナ** — `82m × 104m × 42m` のスタジアム + 斜めコーナー + 縦長ゴール + ブーストパッド 34 枚（大 6 / 小 28）
- 🎨 **3D 描画 (Three.js r128)** — 車・ボール・芝目・観客席・スカイドームをフルでリアルタイム描画
- 🎮 **ジャイロのみで操縦** — 端末を左右に傾けてハンドル、前後傾けてブレーキ。アクセルは自動 ON
- 🆎 **タッチボタンは "JUMP" だけ** — シンプル設計。ブースト用の予備ボタンも一応あり
- 🚀 **ロケットブースター** — ブーストパッドを踏むと燃料が回復、ターボ加速 (空中でも前方推進)
- 🦘 **ジャンプ + ダブルジャンプ** — 空中で 2 回目を押すとフリップ風前方向ダッシュ
- 🎈 **風船みたいなボール** — 重力弱め (公式の半分以下) + 高反発で滞空時間長め、空中プレーが熱い
- 💥 **車の勢いがそのままボールへ** — 反発係数 1.65 + 速度ベクトル転写でめっちゃ飛ぶ
- 🔄 **壁・天井・床に高反発** — 0.78〜0.92 の弾性で立体的なプレーが可能
- 🌐 **PeerJS によるオンライン対戦** — 1v1〜4v4 最大 8 人、部屋コードで合流
- 🤖 **ホストオーソリ方式** — 部屋作成者が物理シミュレーションし 20 Hz で全員に配信
- 🤖 **Bot 練習モード** — 1v1〜4v4 まで CPU 相手にプレー可能 (回り込み / 空中シュート挙動付き)
- ⏱️ **5 分間 1 試合** — タイマー終了時のスコアで勝敗決定、3 カウントダウン付きキックオフ
- 🔊 **SFX 内蔵** — Web Audio API でランタイム生成のヒット音 / ジャンプ音 / ゴール音

## 🕹️ 操作方法 (スマホ)

| 操作 | 入力 |
|------|------|
| 左右ハンドル | 端末を **左右に傾ける** (ジャイロ gamma/beta) |
| アクセル | **自動 ON**（何もしなくても前に進む） |
| ブレーキ / バック | 端末を **手前に傾ける** (ピッチ後傾) |
| ジャンプ / ダブルジャンプ | 画面下中央の `⬆ JUMP` ボタン |
| ブースト | 画面下中央の `🔥 BOOST` ボタン または 設定で「前傾でブースト」を ON |
| ジャイロ感度調整 | 右上 `⚙` アイコン |
| ジャイロ基準リセット | 右上 `⟲` アイコン |
| 試合を抜ける | 右上 `✕` アイコン |

### PC キーボード (デバッグ)

| 操作 | キー |
|------|------|
| ステアリング | ← / → (or A / D) |
| ブレーキ | ↓ (or S) |
| ジャンプ | Space / J |
| ブースト | Shift |

## 🚀 試合の流れ

1. タイトル画面で **名前 / マシンカラー / チーム / マッチサイズ** を選択
2. 以下のいずれかを選ぶ:
   - 🏠 **部屋を作る** → 6 文字の部屋コードが発行されるので友達にシェア
   - 🔑 **部屋に入る** → 6 文字の部屋コードを入力
   - 🤖 **Bot と練習** → ソロモードで Bot と対戦
3. ロビーでチーム (🔵 BLUE / 🟠 ORANGE) を選び、ホストが「**⚽ キックオフ！**」
4. **3, 2, 1, GO!** カウントダウン後試合開始
5. 5 分間の試合（制限時間後にスコアの多いチームが勝利）

## 🛠️ 技術スタック

- **Three.js (r128)** — 3D 描画
- **PeerJS (1.x)** — WebRTC ベースの P2P 通信
- **Web Audio API** — ランタイム SFX 生成 (ライブラリ不要)
- **Vanilla JavaScript** — フレームワーク無しの素な構成

すべてのライブラリは `vendor/` 配下にバンドルされているので、追加のインストール不要。

## 🏃 ローカル起動

`python3` などで静的サーバを立てるだけ:

```bash
git clone https://github.com/guttyanneruuuuuu/soccer.git
cd soccer
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```

> iOS でジャイロを使うには `https://` での配信が必要です（GitHub Pages 等で公開してください）。

## 🤖 ホストオーソリ方式

- **ホスト**（部屋作成者）が、全車の物理シミュレーション・ボール物理・ゴール判定を実行
- 各 **クライアント** は毎フレーム自分の入力 (`steer`, `accel`, `brake`, `boost`, `jump`) をホストへ送信
- ホストは 20 Hz で全車とボールの状態を全員にブロードキャスト
- クライアントは受信状態をローカル予測と Lerp 補間でスムーズに描画

これにより、4v4 = 8 人での同時対戦でも比較的軽量に動作します。

## ⚠️ 既知の制限

- PeerJS の公開クラウドサーバを使用しているため、混雑時に接続が遅延する場合あり
- iOS Safari ではジャイロ利用に明示的な許可が必要（タイトル画面の「許可する」ボタン）
- 端末性能や WebGL ドライバ依存で、ローエンドスマホではフレームレートが低下する場合あり
- 縦持ち時は「横向きにしてね」のオーバーレイが表示されます
