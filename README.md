# ⚽ GYRO SOCCER ARENA

**スマホのジャイロセンサーで操作する、車でサッカーするアリーナ対戦ゲーム。**

Three.js による 3D 描画 + PeerJS による P2P オンライン対戦 + Web Audio API による効果音生成。  
追加アセット不要・ライブラリは vendor 同梱で完全ローカル動作。

> **遊び方:** スマホを横向きにして、左右に傾けるとステアリング、手前(後ろ)に傾けるとブレーキ&バック。
> 画面右下の **ACCEL** ボタンで前進。
> 画面左下の **JUMP** ボタンでジャンプ、二度押しで前方フリップ。

## 🎮 操作

| 操作 | 入力 |
| --- | --- |
| ステアリング | スマホ左右の傾き (ジャイロ gamma/beta) |
| ブレーキ/バック | スマホを手前に傾ける |
| アクセル | 画面右下 ACCEL ボタン |
| ブースト | 画面右側 BOOST ボタン |
| ジャンプ | 画面左下 JUMP ボタン |
| ダブルジャンプ (フリップ) | JUMP 2回連打 — 進行方向にフリップ |
| ハーフフリップ | 前進中に 後ろ傾け + JUMP 2回 → 高速反転 |
| **ハンドブレーキ (DRIFT)** | **画面下中央 ✋ ボタン → パウダースライド** |
| エアロール | 画面左中 AIR ROLL ボタン (空中) |
| クイックチャット | 画面左上の 💬 ボタン |
| カメラ切替 | 画面右上 🎥 (通常/ボール視点) |
| ジャイロ再キャリブレ | 画面右上の ⟲ ボタン |
| 設定 (感度・音量) | 画面右上の ⚙ ボタン |
| リスタート | 画面右上の ↻ ボタン |

PCのキーボード操作にも対応 (デバッグ用):
- 左右矢印 / A / D: ステア
- 下矢印 / S: ブレーキ
- 上矢印 / W: アクセル
- Space / J: ジャンプ
- Shift: ブースト
- E / H: ハンドブレーキ (DRIFT)
- Q / L: AIR ROLL
- V: カメラ視点切替
- R: リスタート

## 🏟️ ゲームモード

- **🤖 Botと練習 (ソロ)** — 1v1 / 2v2 / 3v3 / 4v4 の AI 相手にローカルプレイ
- **🏠 部屋を作る (ホスト)** — 6文字の部屋コードを発行して友達を招く
- **🔑 部屋に入る (クライアント)** — 部屋コードを入力して合流
- 1試合 5分、最大8人(4v4)まで

## 🛡️ パワーアップアイテム (このゲーム独自の追加要素)

フィールド上にランダムに浮遊するアイテムを取ると一定時間バフ:

| アイコン | 名前 | 効果 | 時間 |
| --- | --- | --- | --- |
| 🛡 | **SHIELD** | デモリッション完全無効 | 12秒 |
| 🧲 | **MAGNET** | ボールを引き寄せる | 10秒 |
| 🦏 | **GIANT** | 車1.5倍 & ボール吹き飛ばし 1.7倍 | 8秒 |
| ⚡ | **TURBO** | ブースト無限 | 6秒 |
| 🪀 | **SPRING** | ジャンプ力1.8倍 & 3段ジャンプ | 10秒 |

## 💥 デモリッション

ブースト中の高速車 (56 m/s 以上) で敵チームの車に体当たりすると、相手は爆発して3秒間消失、自陣でリスポーン。
SHIELD パワー所持中はこれを跳ね返せる。

## 🎯 シネマティック追従カメラ

ロケットリーグ系ゲームの肝、車を画面で大きく捉えながらボールも自然に追える可変カメラ:

- 車の **すぐ後ろ・低めから (back=11, up=4.5)** 撮影 → 車が画面いっぱいに迫力
- 速度連動の **動的後退** とブースト時の **FOVキック (68° → 最大83°)**
- 視野内のボールを動的にブレンドして注視 (攻撃中は常にボールが見える)
- 衝突・ハードランディング・ゴール時の **カメラシェイク**

## 🚀 起動方法

ローカルサーバーが必要 (Three.js の WebGL コンテキスト生成のため):

```bash
# Python があれば
python3 -m http.server 8000

# あるいは Node の simple http server
npx http-server -p 8000
```

ブラウザで `http://localhost:8000/` を開く。
スマホで遊ぶ場合は同じネットワーク内の IP で開いて「許可する」を押すこと。

クエリパラメータ `?autosolo=1` を付けると、タイトルをスキップして即ソロ対戦が始まる (デバッグ用)。

## 🧪 開発の流れ (PDCAサイクル)

このリポジトリは複数の PDCA サイクルで段階的に作られている:

1. **PDCA1**: カメラ距離 22→11、車スケール 1.5→1.7、FOVキック、ジャイロ感度UP
2. **PDCA2**: 夜間スタジアム化、ネオン壁・床、ボールトレイル、車下グロー
3. **PDCA3**: デモリッション実装、ゴール40粒子爆発、Bot AI役割分担化
4. **PDCA4**: ミニマップ、クイックチャット、5種パワーアップ
5. **PDCA5**: タイトル/READMEブラッシュアップ、最終バグ修正
6. **PDCA6**: 致命バグ修正・軽量化・新要素追加
   - 🔴 **致命バグ修正**: ゴール後にゲームが固まる根本原因を修正  
     (`PowerUps.reset()` 内で `THREE.Group` に対し `.geometry.dispose()` を呼んでクラッシュしていた問題を再帰的 dispose に置き換え)
   - 🔴 **連続ゴール判定の防止**: ゴール演出中の判定スキップ
   - 🎮 **ジャイロ操作改善**: 非線形カーブ・適応的ローパス・ヒステリシス
   - ⚡ **パフォーマンス**: パーティクルプール / シャドウ自動調整 / クライアント補間
   - 🆕 **新要素**: Bot難易度・試合時間・スローモーション・統計・アシスト

7. **PDCA7 (このアップデート)**: 操作性大強化・新ギミック・大規模軽量化
   - 🆕 **ハンドブレーキ (DRIFT) ボタン**: 画面下中央。押すと急摩擦+ステア倍率1.6倍で
     パワースライド。タイトコーナーで一気に方向転換できる
   - 🆕 **ハーフフリップ**: 前進中に「後傾+ダブルジャンプ」で進行方向と逆向きに反転
   - 🆕 **シールド/MAGNET オーラ可視化**: 取得した瞬間から車に常時アニメオーラが出る
     (ワイヤフレームバリア、磁力線リング)
   - 🆕 **リスタート機能**: 画面右上 ↻ ボタンや終了画面の「もう一回」ボタン
   - 🆕 **試合終了リプレイカメラ**: 全試合終了から 2.5 秒、ボールを高所から旋回撮影
   - 🆕 **強打リング演出**: 60km/h 超のボールヒットで衝撃リング (Three.RingGeometry)
   - 🆕 **感度プリセット**: タイトル & 設定パネルから LOW / MED / HIGH をワンタップで
   - 🆕 **音量スライダー & ミュート**: 設定パネルから即制御
   - 🆕 **コンボ表示派手化**: 4連以上でトースト通知 (`🔥 4x COMBO!`)
   - 🎮 **ジャイロ操作改善**:
     - 二段ステアカーブ (0〜0.4 はかなり緩やか / 0.4〜1.0 で加速して切る)
     - JUMP ボタンに 100ms クールダウンを追加（連打チャタリング防止）
     - マルチタッチで他ボタンを触っても JUMP が誤解放されないように
     - ジャイロ無効時のキーボード入力にイージング適用
   - 🎮 **車挙動チューニング**:
     - 高速時ステア効きを 0.55 → 0.62 にアップ (操作不能感を軽減)
     - 加速 70 → 76 (出だしを少しキビキビ)
     - フリップ終了タイマーを `setTimeout` から物理タイマーに置換
       (スローモーション中も整合)
   - ⚡ **軽量化**:
     - ボールトレイル: 14個 Sphere → 8個 Sprite (頂点コスト 1/3 以下)
     - パーティクルプール: 120 → 60 (静的メモリ削減)
     - パワーアップアイコン Canvas をキャッシュ (毎回生成しない)
     - **シャドウマップ更新を 4 フレに 1 回**に間引き (静的シーンなら問題なし)
     - **ミニマップを 30fps 描画**に間引き
     - **物理を固定タイムステップ 60Hz** に統一 (チャタリング・dt 揺らぎ排除)
     - 円柱・球などのジオメトリの分割数を全体的に削減
   - 🤖 **Bot AI 強化**:
     - パワーアップ箱を見つけたら取りに行く (Normal/Hard)
     - Hard はタイトコーナーで自動でハンドブレーキ発動
     - TURBO/GIANT 取得中は常時ブースト ON
   - 🐛 **バグ修正**:
     - QuickChat バブルが車を毎フレーム追従 (固定位置 → 追従)
     - 試合終了時に車が動き続けるバグを `playLocked` で完全停止
     - `ball.collideWithCar()` の戻り値を最終速度に統一 (一貫性)
     - 設定パネルの UI 配置を整理 (左上 = チャット / 右上 = 設定)

## 📡 マルチプレイ通信 (PeerJS)

P2P ホスト権威モデル:

- **ホスト**: 全車の物理・ボール・ゴール判定を実行。20Hz で状態を全クライアントに配信
- **クライアント**: 自分の入力 (steer/brake/boost/jump) を送信し、ホストから来る状態を補間
- 部屋コードは 6 桁の英数字 (`ABC123` 形式) を発行
- 専用シグナリングサーバー不要 (peerjs.com の公開ブローカーを利用)
